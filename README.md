# AI Powered Chatbot

This project is an AI chatbot built using Python, Flask, and NLP.

## Features
- NLP based chatbot
- Machine learning model
- Flask web interface
- Gemini AI integration

## Technologies Used
- Python
- Flask
- NLTK
- TensorFlow / Keras
- HTML, CSS, JavaScript

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the project:

python app.py

Open browser:

http://127.0.0.1:5000
